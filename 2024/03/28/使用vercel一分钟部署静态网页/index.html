使用vercel快速部署静态网页

最近有一个简单的Web应用需要上线，尝试使用vercel。神仙网站www

  1. 进入vercel官网
  2. 登陆后自动进入dashboard

  3. add new … -> project

  4. import 我们要部署的静态网页项目（需要是公有项目，如果还没有显示就点击 Adjust Github APP Permissions）

  5. 配置项目（若index.html不在根目录下，则需要修改 Root Directory）

  6. 点击Deploy，很快，我的小项目不到10s就好了

    显示比例有点奇怪，不知道为什么
  7. 点击预览图片即可进入网站。回到dashboard可以查看网址，可以添加其他域名
